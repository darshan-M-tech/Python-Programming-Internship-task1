Project Title:

Internship Task 1 – User Info Display

Description:

This is the first task of the Python programming internship at Elevate Lab.
The purpose of this task is to create a simple Python program that displays:

Username

Internship Role

Today’s Date

This task helps in understanding basic Python syntax, printing output, and using the datetime module.

Features:

Displays the username of the intern

Shows the assigned internship role

Prints the current date in a readable format

Prerequisites:

Python 3.x installed on your machine

Basic knowledge of Python (variables, print statements, modules)

How to Run:

Clone or download the project files.

Open the terminal/command prompt.

Navigate to the project folder.

Run the program:

python task1_user_info.py

Sample Output:
Username: Darshan
Internship Role: Python Programming Intern
Today's Date: 2026-01-15

Code Snippet:
# hello_world.py

from datetime import date

username = "Darshan"
internship_role = "Python Programming Intern"
today_date = date.today()

print("Username:", username)
print("Internship Role:", internship_role)
print("Today's Date:", today_date)
